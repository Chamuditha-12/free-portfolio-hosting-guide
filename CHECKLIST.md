# ✅ Quick Deployment Checklist

## Before You Start
- [ ] Portfolio code ready on GitHub
- [ ] Have a payment method for domain

## Domain (Spaceship)
- [ ] Visit spaceship.com
- [ ] Search domain name
- [ ] Apply promo code: `COMPROS`
- [ ] Complete purchase (~$2.90)

## Hosting (Netlify)
- [ ] Create free Netlify account
- [ ] Connect GitHub repo
- [ ] Deploy site

## DNS Connection
- [ ] Add custom domain in Netlify
- [ ] Copy Netlify nameservers
- [ ] Update nameservers in Spaceship
- [ ] Wait 24 hours

## Done!
- [ ] Visit your domain 🎉

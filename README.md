## ☁️ Step 2 — Set Up Free Hosting (Netlify)

1. Go to [netlify.com](https://netlify.com)
2. Sign up for a **free account**
3. Click **"Add new site"** → Deploy from GitHub
4. Connect your GitHub repo (your portfolio code)
5. Click **Deploy site** ✅

---

## 🔗 Step 3 — Connect Domain to Netlify

1. In Netlify → go to **Site Settings → Domain Management**
2. Click **"Add a custom domain"**
3. Enter your domain name (e.g. `yourname.com`)
4. Netlify will show you **nameservers** (e.g. `dns1.p01.nsone.net`)
5. Go to **Spaceship dashboard → DNS Settings**
6. Replace default nameservers with the ones from Netlify
7. Wait up to **24 hours** for DNS propagation ✅
